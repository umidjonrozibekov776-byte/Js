🟨 JavaScript — bu nima?

JavaScript (JS) — bu veb sahifalarga interaktivlik qo‘shish uchun ishlatiladigan dasturlash tili. U dastlab faqat brauzerlar ichida ishlatilgan, ammo bugungi kunda uni serverlar, mobil ilovalar, desktop dasturlar, va hatto sun'iy intellekt loyihalarida ham ishlatish mumkin.




📜 JavaScriptning qisqacha tarixi:


1995 — Brendan Eich (Netscape kompaniyasi) JavaScript’ni yaratdi.
1996 — Microsoft JavaScript’ning o‘z versiyasini yaratdi — JScript.
1997 — ECMA standardi sifatida tan olindi (ECMAScript).
2009 — Node.js taqdim etildi (JS endi serverda ham ishlay boshladi).
2015 — ES6 standarti chiqarildi (JS’da katta o‘zgarishlar bo‘ldi: let, const, arrow functions, classes, va h.k.).
